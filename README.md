# 岭风译游 - 广东跨境文旅服务创新平台

## 项目简介

"岭风译游"是一个面向境外游客、在粤外籍人士及海外文旅爱好者的中英双语文旅服务平台。围绕广东/岭南文化内容展示、旅游路线推荐、陪游口译预约、文创商品展示与销售、广告合作等业务，形成基础商业闭环。

## 技术栈

- **框架**: Next.js 14 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **动画**: Framer Motion
- **状态管理**: Zustand
- **国际化**: 自定义 i18n 方案

## 功能模块

### 前台页面
- 首页 - 品牌展示、轮播Banner、热门推荐
- 文化专题页 - 广府/潮汕/客家/非遗/美食
- 路线列表与详情页
- 陪游口译预约页
- 文创商城页
- 广告合作页
- 用户反馈页
- 联系我们页

### 核心特性
- 中英双语切换
- 响应式设计 (PC/移动端)
- 流畅动画效果
- 岭南文化特色视觉设计

## 开发指南

### 安装依赖

```bash
npm install
# 或
yarn install
# 或
pnpm install
```

### 启动开发服务器

```bash
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000) 查看效果。

### 构建生产版本

```bash
npm run build
npm run start
```

## 项目结构

```
src/
├── app/           # 页面路由
├── components/    # 组件库
├── lib/           # 工具函数
├── hooks/         # 自定义Hooks
└── store/         # 状态管理
```

## 配色方案

项目采用岭南文化特色配色：

- **主色**: 岭南红 (#ee7510) - 取自岭南建筑、粤绣金线
- **辅助色**: 翡翠绿 (#14b8a6) - 取自岭南园林、满洲窗
- **点缀色**: 紫荆紫 (#d946ef) - 取自香港市花
